作品タイトルLIBERAR (ファッション通販サイト)
制作日2020年11月16日
制作背景 ペルソナを意識したコンセプトに沿ったデザインと、コーディングスキルを鍛える目的
解説 ファッションブランド『LIBERAR』という架空の通販サイトです。ペルソナ設定に対応させるため、きれいめな英字フォントとフォーマルな写真で高級感を出し、花の鮮やかでヴィンテージなカラーを取り入れ独創的なファッションを表現しています。また、訪問者の購買意欲を高めるためにセール情報を強調し、雑誌に有名女優（架空）を起用することでブランドへの関心や認知度を高めるようアピールしています。
制作目的 ・フォーマルで、流行より自分らしい個性的なファッションを求める人へ、最適なブランドである事をアピール
・通販での売り上げ増加
制作日数約3週間
（ペルソナ設定・ワイヤーフレーム：２週間、デザイン・コーディング：１週間）
学んだこと・静的なコーディングスキル
・ユーザビリティを意識した彩度・明暗の調整と、ブランドコンセプトに基づく雰囲気作りを共有させるデザインスキル
ペルソナ ・20代から30代半ばの男女
・都内在住の独身
・月に１万円~５万円をファッションに投資
・ヴィンテージかつフォーマルな雰囲気を好み、他人と被らない個性的な服を求めるという何よりのこだわりを持つ
使用ツールPowerPoint / Photoshop / HTML / CSS
作品タイトルdot.y
(ファッション通販サイトのファーストビュー)
制作日2020年10月18日
制作背景 ブランドの系統を一目で認識できるデザインスキルを鍛える目的
解説ファションブランド『dot.y』という架空の通販サイトです。フォントの統一感や背景写真を含めた全体的な配色を白黒金にまとめることで、シンプルかつラグジュアリーな雰囲気を作っています。また背景写真をトップで大きめに配置しているところには、ブランドのイメージをファーストビューで掴んでもらうためという意図が込められています。
制作目的 ・高級感のあるモードなファッションをアピール
・通販での売り上げ増加
制作時間約5時間
学んだこと ・ファーストビューのみでブランドの系統を認識させるデザインスキル
・強弱をつけた色使い・空間構成
使用ツールPhotoshop
作品タイトルRUI (音楽サイトのファーストビュー)
制作日2020年　11月1日
制作背景 背景を強調しつつ、フォントや文字配置の調整によって情報をわかりやすく伝えるスキルを鍛える目的
解説アーティスト『RUI』という架空の音楽サイトです。挿入文字の色やフォントを統一し写真と共にクールな印象を作りつつ、アルバム情報を空白のバランスを取りながら配置したことで、アルバムの雰囲気や情報も掴みやすいシンプルなサイトに仕上げました。
制作目的・クールかつアンニュイな雰囲気の演出
・アルバムの雰囲気を壊さず、かつわかりやすい告知で購買率増加
制作時間約3時間
学んだこと ・アルバムジャケットを強調しつつ、情報をわかりやすく伝えるデザインスキル
使用ツールPhotoshop
タイトル DEPP'S MUSIC EVENT
(音楽イベントのフライヤー)
制作日2020年10月23日
制作背景 テーマという抽象的なものを、具体的なイラストで表現するスキルを鍛える目的
解説DJ『DEPP』による「世界平和」がテーマである架空の音楽イベントのフライヤーです。「世界平和」がテーマのため、「花びらが揃ってこそ美しく見える花のように、各国が平和で一つになり、美しい世界になりますように」という想いを込めて、オリンピックの５大陸を表した５輪カラーをモチーフにして、同じ５色の花びらを持つ花を中心に大きく配置しました。また、中心の花が目立つよう背景色と文字色を無彩色にし、フォントまで揃えることで花だけでなくイベント情報まで見やすくなるよう工夫しました。
制作目的イベントにおけるコンセプトデザインを用いた、わかりやすいフライヤーで集客率増加
制作時間約8時間
学んだこと ・シンプルで具体的なものを、抽象的な意味を持つものとしてイラストするデザインスキル
使用ツールPhotoshop / Illustrator
タイトル 福善屋敷
(和カフェサイトのファーストビュー)
制作日2020年10月29日
制作背景 シンプルで見やすく、主役を目立たせるサイト構成にするためのデザインスキルを鍛える目的
解説和カフェ『福善屋敷』の架空のコーポレートサイトです。挿入文字に堅い印象のフォントを使用し、ロゴを筆のフォントでハンコ風にすることで和風の雰囲気を作り出しています。また、写真の周りをほぼ無彩色で囲むことで、写真のメインであるお茶が映えるようデザインしています。
制作目的 お店の高級感のある和風というイメージを一目で認識させるため
制作時間約7時間
学んだこと シンプルかつ最も目立たせたいお茶の部分に視線を誘導させる空間構成と色使いを意識したデザインスキル
使用ツールPhotoshop / Illustrator
タイトル ANSWER
(美容院サイトのファーストビュー)
制作日2020年10月19日
制作背景 強弱のある空間構成と色使いで、サイトがユーザーに伝えたい部分と見やすさを表現するデザインスキルを鍛える目的
解説美容院『ANSWER』という架空のコーポレートサイトです。このお店のコンセプトは「個性」のため、それを表すカラフルな男女の写真と、信頼感のあるシンプルな内装写真を配置。また、より訪問者を惹きつけるため、お店が最も大切にしているコンセプトのメッセージを簡潔にまとめ、配色は有彩色をメニューバーの赤のみにすることで、全体的にシンプルで見やすいサイトに仕上げました。
制作目的・予約に誘導し集客率増加
・男女ともに個性派まで対応可能な信頼感のあるお店をアピール
制作時間約4時間
学んだこと 強弱のある空間構成と色使いで強調したい部分を表現するデザインスキル
使用ツールPhotoshop
